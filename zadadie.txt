zadanie
git init - создание репозитория
git status - просмотр файлов в репозитории
git add . или git add -A - добавить все файлы в репозиторий ( git add namefile - один файл)
git commit -m "сообщение коммита"


ветвление
git branch <имя новой ветки> - создание ветки
git checkout -b <имя_новой_ветки> - переход на ветку
git branch - подтвердить, что ветка создана
git add <имя_нового файла в новой ветеке>.txt 
git commit -m "New имя_новой_ветки complete - коммит создания новой ветки
git checkout <имя основной ветки> - переход в основную ветку 

git merge <имя ветки с изменениями> - слияние веток
git merge --abort  - остановить слияние веток
git branch -d <имя ветки  с изменениями> - удаление ветки с изменениями
git branch -D <имя ветки  с изменениями> - подтверждение удаление ветки с изменениями
git push - применение всех изменений
